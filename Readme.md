# Descubra Guapi – API de Turismo Ecológico

API desenvolvida para promover o turismo ecológico em Guapimirim, permitindo o cadastro, avaliação e gerenciamento de pontos turísticos como trilhas, cachoeiras e mirantes.

## Entidades Principais

### Pontos Turistico
Representa um local turístico disponível no sistema, como trilhas, cachoeiras, mirantes ou campings.  
**Rotas principais:** `/pontos`  

### Usuario
Usuários que podem se cadastrar, acessar o sistema e realizar avaliações.  
**Rotas principais:** `/usuarios`  

### Avaliacao
Permite que os usuários avaliem os pontos turísticos, deixando notas e comentários.  
**Rotas principais:** `/avaliacoes`  

## Objetivo
Facilitar a descoberta e valorização dos atrativos naturais de Guapimirim, incentivando o turismo sustentável e a interação entre visitantes e natureza.
